# GameVault – colección de videojuegos

App Flutter con Material 3 que permite gestionar tu biblioteca personal de juegos, favoritos y categorías.

## Estructura del proyecto 
lib/

├── main.dart   

├── screens/

│   └── home_screen.dart   Pantalla principal con tabs

└── widgets/

├── app_drawer.dart        Menú lateral (Drawer)

├── games_grid.dart        Cuadrícula de juegos

└── categories_list.dart   Lista de categorías

## Ejecuta el proyecto
flutter run
